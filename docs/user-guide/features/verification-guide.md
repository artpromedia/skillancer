## Taking Assessments

### Finding Assessments

1. Go to **Profile â†’ Skills & Expertise**
2. Click **"Get Verified"** on any skill
3. Or visit **Verification Center** from main menu

### Assessment Categories

| Category               | Skills Tested                                    |
| ---------------------- | ------------------------------------------------ |
| **Web Development**    | JavaScript, React, Node.js, HTML/CSS, TypeScript |
| **Mobile Development** | React Native, Flutter, iOS, Android, Swift       |
| **Backend**            | Python, Java, Go, Ruby, PHP, C#, Node.js         |
| **DevOps**             | AWS, Docker, Kubernetes, CI/CD, Terraform        |
| **Data Science**       | Python, SQL, Machine Learning, Statistics        |
| **Design**             | UI/UX, Figma, Adobe Creative Suite               |
| **Project Management** | Agile, Scrum, PMP concepts                       |

### Assessment Structure

#### Standard Assessment

- **Duration**: 15-45 minutes
- **Questions**: 25-50 items
- **Format**: Multiple choice, code snippets
- **Passing Score**: 70%

#### Advanced Assessment

- **Duration**: 60-90 minutes
- **Questions**: 15-25 items
- **Format**: Coding challenges, system design
- **Passing Score**: 75%

### Preparation Tips

1. **Review fundamentals** before taking assessment
2. **Time management**: Don't spend too long on one question
3. **Read carefully**: Questions may have tricky wording
4. **Use scratch paper**: For calculations and diagrams
5. **Stay calm**: You can retake after 30 days

### During the Assessment

**Rules:**

- No external resources (unless open-book assessment)
- No screen sharing or recording
- Complete in one session
- Stable internet required

**Environment:**

- Full-screen mode required
- Browser tab switching monitored
- Webcam verification (some assessments)

### Assessment Results

After completion, you'll see:

- **Overall Score**: Percentage correct
- **Pass/Fail Status**: Based on threshold
- **Skill Ranking**: Your percentile vs. others
- **Strengths/Weaknesses**: Topic breakdown
- **Badge Earned**: If passed

### Retake Policy

| Scenario              | Wait Time |
| --------------------- | --------- |
| Failed assessment     | 30 days   |
| Want to improve score | 90 days   |
| Assessment updated    | Immediate |

### Linking Certifications

1. Go to **Profile â†’ Credentials**
2. Click **"Add Credential"**
3. Select certification type
4. Enter:
   - Certificate ID/Number
   - Issue date
   - Expiration date (if applicable)
   - Issuing organization
5. Upload certificate image (optional)
6. Click **"Verify"**

### Verification Process

For most certifications:

1. **Automatic**: We check issuing authority database
2. **Manual**: Support team verifies if auto-check unavailable
3. **Timeline**: Usually within 24-48 hours

### GitHub Integration

Connect GitHub to verify coding skills:

1. Go to **Credentials â†’ Connect GitHub**
2. Authorize Skillancer access (read-only)
3. We analyze:
   - Contribution history

   - Open source contributions

### LinkedIn Integration

Import professional credentials:

### Education Verification

Verify degrees and diplomas:

## Getting Endorsements

### What Are Endorsements?

Endorsements are recommendations from:

- Past clients
- Colleagues
- Other Skillancer users
- External contacts

### Types of Endorsements

#### Client Reviews

Automatically created after contract completion:

- **Star Rating**: 1-5 stars
- **Written Review**: Client's feedback
- **Skill Endorsement**: Skills used on project
- **Private Feedback**: Not publicly visible

#### Skill Endorsements

Quick endorsements for specific skills:

- Click-to-endorse on profile
- Count shown on each skill
- Higher count = more credibility

#### Recommendations

Detailed written recommendations:

- Longer form than reviews
- From colleagues or managers
- Can be from outside Skillancer

### Requesting Endorsements

#### From Past Clients

1. Go to **Contracts â†’ Past Contracts**
2. Find contract without review
3. Click **"Request Review"**
4. Personalize request message
5. Send reminder if needed

#### From Colleagues

1. Go to **Profile â†’ Endorsements**
2. Click **"Request Endorsement"**
3. Enter colleague's email
4. Customize request
5. They receive link to submit

### Endorsement Best Practices

**Do:**

- âœ… Request soon after project completion
- âœ… Make it easy with personalized requests
- âœ… Thank those who endorse you
- âœ… Endorse others to build relationships

**Don't:**

- âŒ Spam requests
- âŒ Offer incentives for endorsements
- âŒ Create fake endorsements
- âŒ Harass non-responders

### Managing Endorsements

#### Visibility Control

- Choose which reviews are featured
- Hide inappropriate reviews (with reason)
- Respond to reviews publicly

#### Responding to Reviews

1. Navigate to review
2. Click **"Respond"**
3. Write professional response
4. Response visible to all

**Response Tips:**

- Thank positive reviewers
- Address concerns professionally
- Stay calm with negative feedback
- Offer to make things right

### Review Disputes

If a review is unfair:

1. **Respond Publicly**: Share your perspective
2. **Report Violation**: If review violates policies
3. **Request Mediation**: Support team can mediate
4. **Accept Outcome**: Reviews removed only for policy violations

## Verification FAQ

**Q: Are assessments free?**
A: Yes, all Skillancer assessments are free. You can retake after 30 days.

**Q: How long do verifications last?**
A: Skill assessments don't expire. External certifications expire when the underlying credential expires.

**Q: Can I verify skills not listed?**
A: Request new assessments through the Verification Center. We add new skills regularly.

**Q: Do verifications transfer from other platforms?**
A: External certifications can be added. Platform-specific tests must be retaken.

**Q: How do I become Expert-Vetted?**
A: Apply through the Verification Center. Requires interview and work sample review.
