## Creating Your Account

### Step 1: Sign Up

1. Visit [skillancer.com](https://skillancer.com) and click **"Hire Talent"**
2. Choose your account type:
   - **Individual**: For personal projects
   - **Business**: For company hiring (includes team features)
3. Enter your email and create a secure password
4. Verify your email address

### Step 2: Complete Your Profile

A complete client profile attracts better talent:

- **Company/Individual Name**: Your display name
- **Logo/Photo**: Builds trust with freelancers
- **Description**: Brief overview of your business
- **Location**: Helps with timezone matching
- **Payment Method**: Required before hiring

### Step 3: Verify Your Account

Verified clients get **40% more proposal responses**:

1. Add a valid payment method
2. Verify your phone number
3. Complete identity verification (for higher limits)
4. Add company verification (optional, for businesses)

## Posting Your First Job

### Choosing Job Type

| Type            | Best For                                      | Payment             |
| --------------- | --------------------------------------------- | ------------------- |
| **Fixed-Price** | Well-defined projects with clear deliverables | Pay per milestone   |
| **Hourly**      | Ongoing work or exploratory projects          | Pay for time worked |

### Writing an Effective Job Post

#### Title

Be specific and keyword-rich:

**Good Examples:**

- "React Developer Needed for E-commerce Dashboard - 3 Month Project"
- "UI/UX Designer for Mobile Banking App Redesign"
- "DevOps Engineer to Set Up CI/CD Pipeline on AWS"

**Avoid:**

- "Need developer ASAP!!!"
- "Looking for help"
- "Website project"

#### Description

Include these sections:

1. **Project Overview** (2-3 sentences)
   - What is the project about?
   - What is the end goal?

2. **Scope of Work**
   - Specific tasks and deliverables
   - Technical requirements
   - Timeline expectations

3. **Required Skills**
   - Must-have qualifications
   - Nice-to-have skills
   - Experience level needed

4. **About Your Company** (optional but recommended)
   - Brief background
   - Why this project matters

#### Budget Setting

**Fixed-Price Projects:**

- Research market rates for similar projects
- Consider complexity and timeline
- Be realistic‚Äîlowball budgets attract inexperienced talent

**Hourly Projects:**

- Set an hourly range (e.g., $40-60/hour)
- Estimate weekly hours needed
- Define project duration

#### Skill Tags

Select relevant skills for better matching:

- Add 3-5 primary skills
- Include specific technologies
- Avoid overly broad tags

### Job Visibility Options

| Option          | Description                            |
| --------------- | -------------------------------------- |
| **Public**      | Visible to all freelancers             |
| **Invite-Only** | Only invited freelancers can apply     |
| **Private**     | Hidden from search, shareable via link |

## Hiring a Freelancer

### Making an Offer

1. Click **"Hire"** on the freelancer's proposal
2. Configure contract terms:
   - **Payment Type**: Fixed or hourly
   - **Weekly Limit**: Maximum hours (hourly)
   - **Start Date**: When work begins
   - **Milestones**: For fixed-price projects

### Contract Types

#### Fixed-Price Contracts

- Define milestones with clear deliverables
- Fund escrow for each milestone
- Release payment upon approval
- Best for projects with defined scope

**Example Milestone Structure:**
| Milestone | Deliverable | Amount |
|-----------|-------------|--------|
| 1 | Design mockups | $500 |
| 2 | Frontend development | $1,500 |
| 3 | Backend integration | $1,500 |
| 4 | Testing & launch | $500 |

#### Hourly Contracts

- Set weekly hour limits
- Review work diary screenshots
- Pay for actual time worked
- Best for ongoing work

### Escrow Protection

All payments are protected:

1. You fund the milestone/weekly limit
2. Funds are held in escrow
3. Freelancer completes work
4. You approve and release payment
5. Funds transfer to freelancer

## Managing Contracts

### Communication Best Practices

- **Be Responsive**: Reply within 24 hours
- **Be Clear**: Provide detailed feedback
- **Be Available**: Schedule regular check-ins
- **Use Platform**: Keep communication on Skillancer

### Milestone Management

#### For Fixed-Price:

1. Freelancer submits milestone
2. Review deliverables thoroughly
3. Request revisions if needed
4. Approve to release payment

#### Approval Timeline:

- Auto-approval after 14 days if no response
- Request extensions if you need more time
- Dispute if deliverables don't meet requirements

### Hourly Work Tracking

Skillancer's Work Diary captures:

- Screenshots at regular intervals
- Activity levels (keyboard/mouse)
- Memos describing work done
- Time logged per day

**Reviewing Work Diary:**

1. Go to **Contracts ‚Üí Work Diary**
2. Review time entries weekly
3. Dispute inaccurate time within 5 days
4. Approved time is paid automatically

### Using SkillPod

For high-security projects:

1. Enable SkillPod requirement on contract
2. Freelancer works in secure environment
3. Sessions are recorded for compliance
4. Access recordings for audit purposes

### Contract Modifications

You can modify active contracts:

- Increase/decrease weekly limit
- Add new milestones
- Extend timeline
- Adjust bonus terms

## Processing Payments

### Payment Methods

Add a billing method before hiring:

| Method             | Availability | Fees   |
| ------------------ | ------------ | ------ |
| Credit/Debit Card  | Instant      | 2.75%  |
| Bank Account (ACH) | 3-5 days     | Free   |
| PayPal             | Instant      | 3%     |
| Wire Transfer      | 1-3 days     | Varies |

### Payment Schedule

| Contract Type | When Charged          |
| ------------- | --------------------- |
| Fixed-Price   | When milestone funded |
| Hourly        | Weekly (every Monday) |

### Understanding Fees

**Client Service Fee**: 3% of all payments

This covers:

- Payment protection
- Dispute resolution
- 24/7 support
- Platform maintenance

### Invoicing & Reports

Access financial data in **Cockpit Dashboard**:

- **Transactions**: All payment history
- **Invoices**: Downloadable for accounting
- **Reports**: Spending by project/freelancer
- **Tax Documents**: Annual summary (1099 for US)

### Disputes & Refunds

If issues arise:

1. **Communicate First**: Most issues resolve with discussion
2. **Request Revision**: Give freelancer chance to fix
3. **Open Dispute**: If unresolved after good-faith effort
4. **Mediation**: Skillancer mediates disputes
5. **Arbitration**: Final resolution if needed
