# Billing Service

Handles billing, payments, subscriptions, and invoicing.

## Tech Stack

- **Runtime**: Node.js 20+
- **Framework**: Fastify
- **Language**: TypeScript
- **Payment Provider**: Stripe

## Getting Started

```bash
# From monorepo root
pnpm dev --filter=@skillancer/billing-svc

# Or from this directory
pnpm dev
```

## Features

- Subscription management
- One-time payments
- Invoice generation
- Payment method management
- Usage-based billing
- Webhook handling
