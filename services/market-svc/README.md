# Market Service

Core marketplace business logic for the Skillancer talent marketplace.

## Tech Stack

- **Runtime**: Node.js 20+
- **Framework**: Fastify
- **Language**: TypeScript
- **Database**: PostgreSQL (via Prisma)

## Getting Started

```bash
# From monorepo root
pnpm dev --filter=@skillancer/market-svc

# Or from this directory
pnpm dev
```

## Features

- Talent profiles and discovery
- Job/project posting
- Bidding and proposals
- Matching algorithms
- Reviews and ratings
- Messaging integration
