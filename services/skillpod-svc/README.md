# SkillPod Service

VDI (Virtual Desktop Infrastructure) management and orchestration service.

## Tech Stack

- **Runtime**: Node.js 20+
- **Framework**: Fastify
- **Language**: TypeScript
- **Infrastructure**: Docker / Kubernetes

## Getting Started

```bash
# From monorepo root
pnpm dev --filter=@skillancer/skillpod-svc

# Or from this directory
pnpm dev
```

## Features

- VM/Container provisioning
- Session management
- Resource allocation
- Streaming configuration
- Security policies
- Usage monitoring
